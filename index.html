BetterRecycling
Project

BetterRecycling

Project



Using convolutional neural networks in PyTorch to classify waste!

Summary
Using convolutional neural networks in PyTorch to classify waste! Using CNNs I created, trained and deployed a system onto a Raspberry Pi, to automatically classify everyday objects into four different categories (plastic, paper, glass and metal). I also wrote a paper detailling my work and explaining underlying concepts as well as discussing modelling choices.
Dataset
~2500 images were manually collected and categorized. I received support from family and friends for collecting the data. If you are interested in the dataset, it is publically available! You can download it on my GitHub.
Further Use
This project was presented on a multiple events, such as the Bosch AI CON or the Stuttgart Future Symposion. Currently I am creating an Open Source-course for the BWKI, where I explain the basics of convolutional neural networks and show participants, how they could have built my machine learning project from scratch!
Technologies
  • Python 3
  • PyTorch, torchvision
  • NumPy, Matplotlib, PIL
  • Raspberry Pi

Awards
With this project I reached the national final (top 8) of the German National Contest for AI ([Bundeswettbewerb Künstliche Intelligenz]) in 2019 with over 2500 participants in total.

Neural Networks From Scratch
Project

Implemeting neural networks from scratch!

Summary
Feedforward neural networks using only NumPy! This was my "Facharbeit" for my cs honours class in eleventh grade. You can create, train and evaluate (with plots!) neural networks of any shape. An example architecture reached approx. 95% test accuracy on MNIST after one training epoch. Hierzu wurde eine vollständige Facharbeit verfasst, welche den gesamten Entwicklungsprozess erklärt.
Technologies
  • Python 3
  • NumPy

Awards
The Facharbeit received the Dr.Hans-Riegel award (see here for outstanding scientific work ("herausragende wissenschaftliche Arbeit") in 2019. It ranked 2nd place in computer science at the corresponding contest at the Johannes Gutenberg University in Mainz, which represented the state of Rheinland-Pfalz.

Features
  • It is possible to save and load network models.
  • You can view the training-accuracy of your network over time graphically.
  • You can draw your own digits and classify them using your network!

Fine Dust Simulation
Project

A program to simulate the creation and spreading of fine dust: fiduwisi!

Summary
Fiduwisi, or "fine dust - wind simulation", is a small program, which simulates and visualizes the creation and spreading of finde dust in the air, or air pollution in general simulieren. A 1-minute video-summary of the project can be viewed above

Technologies
  • Python 3
  • tkinter

Awards
This program was included in the categorie of "outstanding entries" of a Software-Engineering-Contest with 77 participants.

Features
This program...
  • shows the current state of the simulation in real-time using a cellular automaton.
  • randomly generates the simulation map
  • generates fields, villages, as well as small and large cities.
  • generates winds with different properties such as wie width, length or wind force.
  • offers the ability to view the state of a single large city complex over time in a real-time graph-plot..
  • offers a variety of settings where f.e. the wind frequency can be adjusted.
  • contains a complete tutorial, description and explanation of the mechanics of fiduwisi.

A program for calculating the "Abiturschnitt": The MSS-Calculator!

Summary
MSS-rechner is a small programm designed for students of the german "Mainzer Studienstufe" (MSS, 11th-13th grade). This program enables them to quickly and comfortably calculate their final graduation score, even if they do not have all of the required grades yet. In the latter case a forecast is calculated.

Technologies

  • tkinter


Features
This program...
